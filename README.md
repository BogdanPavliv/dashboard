## Верстка сайту Dashboard

- git,
- gulp,
- npm,
- figma, svg where it is possible,
- html,
- scss,
- javascript,
- flex boxes,
- BEM naming

Documentation and usage examples inside.

Install dependencies:
```
npm i
```

Run in development mode:
```
gulp
```

To create production version use:
```
gulp building
```
